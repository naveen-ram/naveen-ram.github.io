Retirement locator

This project was completed for the IBM Coursera Data Science Capstone. It uses various python based tools such as pandas, folium, sklearn, numpy, as well as data collected from public foursquare and weather APIs to provide retirees with suggestions on where to retire, using kmeans clustering on the city data.

ML Autosegmentation of Brain Tumors

I completed this project for EBME 361, a course on image analysis at Case Western. In this project, I used feature extraction, training, testing, cross validation, evaluation techniques in MATLAB. I used T1-weighted MRIs of patients injected with Gadolinium contrast provided by the instructors of the course.
